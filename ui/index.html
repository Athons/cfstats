Tech Details

This page is built from the code in athons/cfstats, which uses:

  • Python for talking to the Cloudflare API and downloading the data.
  • Chart.js for Graphs
  • Splendor for some nice default CSS

Check the code out if this sort of thing interests you, and feel free to submit patches to make improvements!
