Athon stats!

This are directly fetched from the cloudflare API and might be a little inaccurate (from bots to vpns, etc)

This page is built from the code in athons/cfstats, which uses:

  • CF data fetching is done in Python
  • Graphs made with Chart.js
  • CSS theme is Splendor

Check the code out if this sort of thing interests you, and feel free to submit patches to make improvements!

Data is collected by a script which reads the last weeks worth of data, daily.
